/* 原理：固定宽度，利用left距离位移。 */

body{
    background: #2c3e50;
}

nav{
    position: relative;
    margin: 260px auto 0;
    width: 620px;
    height: 50px;
    background-color: #34495e;
    border-radius: 8px;
    font-size: 0;
    box-shadow: 0 2px 3px 0 rgba(0,0,0,.1);
}

nav a{
    font-size:15px;
    text-transform: uppercase;
    color: white;
    text-decoration: none;
    line-height: 50px;
    position: relative;
    z-index: 1;
    display: inline-block;
    text-align: center;
}

nav .animation{
    position: absolute;
    height: 100%;
    /* height: 5px; */
    top: 0;
    /* bottom: 0; */
    z-index: 0;
    background: #1abc9c;
    border-radius: 8px;
    transition: all .5s ease 0s;
}

/* hover就是悬浮效果而已鼠标移开就会回到原位置，start-about这些是预备给跳转之后的页面nav初始状态用的 */

a:nth-child(1){
    width: 110px;
}

nav .start-home,a:nth-child(1):hover~.animation{
    width: 110px;
    left:0;
}

a:nth-child(2){
    width: 120px;
}

nav .start-about,a:nth-child(2):hover~.animation{
    width: 120px;
    left:110px;
}

a:nth-child(3){
    width: 110px;
}

nav .start-blog,a:nth-child(3):hover~.animation{
    width: 110px;
    left:230px;
}


a:nth-child(4){
    width: 160px;
}

nav .start-portfolio,a:nth-child(4):hover~.animation{
    width: 160px;
    left:340px;
}

a:nth-child(5){
    width:120px;
}

nav .start-contact,a:nth-child(5):hover~.animation{
    width: 120px;
    left:500px;
}

nav标签 
下的 
start-contact类选择器 

和 

a标签父元素下的第5个子元素 在鼠标悬浮元素上方时

a标签的的兄弟--animation类选择器以及animation类选择器之后的选择器


的样式
