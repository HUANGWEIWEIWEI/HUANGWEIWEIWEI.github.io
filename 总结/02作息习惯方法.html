# 如何早睡、早起

  1. 21:30以后不要再看电子产品。

  2. 22:00 停止所有事情。

  3. 睡不着就起来换个地方看会儿书。起来换个地方做冥想、读书、写字等比较安静的一些事情，让大脑也逐渐放松下来。

  4. 设置一个闹钟,坚决不睡回头觉。

  5. 早起的关键就是为自己 制定目标计划，唤醒意识 。

  6. 尽量少喝咖啡，下午2点以后就不要再喝了，也不要每天都喝，床单要一周一换。

# 如何健康减肥

  1. 早餐多吃点，养胃。中餐吃的营养些。晚餐不吃。超过下午3点只喝水不吃东西。

  2. 高抬腿减肚子。慢跑减全身。

  3. 少吃多动。

  4. 不喝饮料。不吃垃圾食品。减少不必要的盐的摄入，主餐要清淡。每天吃水果。

# 如何对学习上瘾

  1. 瘾是一种让你感受更好的机制。
  2. 上瘾原则：随机奖励、获取方便、没有尽头
  3. 知识≠智慧
  4. 反向拆解那些令人上瘾产品的机制：①可望不可即的目标、②无法抵挡与预知的积极反馈、③渐进式的进步和改善的感觉、④随着时间的推移越来越困难的任务、⑤需要解决但又尚未解决的紧张感、⑥强大的社会联系。
  5. 有些上瘾的东西虽然能让你得到短暂性的满足，但对你长久的发展是无用甚至不利的。

如何把瘾转移到学习上。享受学习。

  1. 可望不可即的目标：招聘软件上看看人外有人的薪酬
  2. 无法抵挡与预知的积极反馈：用所学知识做小项目
  3. 渐进式的进步和改善的感觉：以数值或者笔记的方式记录学习成果
  4. 随着时间的推移越来越困难的任务：尝试深入学习底层原理
  5. 需要解决但又尚未解决的紧张感：尝试拓宽学习面
  6. 强大的社会联系：群交流、刷社交平台、认识更多同行

# 学习方法

  1. 💡把更多的注意力和时间牵引到学习上面
  2. ✨信念：选择自律、选择延迟满足、选择会让你不那么舒服的努力和成长。
  3. 🚫不可疲劳低效学习：每隔半小时左右切换一下学习状态：看电脑学习/看手机学习/看书学习/做项目/做笔记/刷算法题/刷面经/刷学习性质的社交平台

# 视力恢复

有意识地控制眼周肌肉压缩眼球

# 第一阶段 视觉恢复闪现

工作看书或者用电脑的时候，尽量保持在看清和看不清的临界状态

# 第二阶段 放大闪现效果阶段

此阶段目标：做到瞬间睁眼后，能够在让眼前保持一段时间的清晰。

当你有足够多的瞬间清晰闪现经历后，可以尝试放大这种感觉。这个阶段就可以进行练习了。具体做法如下：

可以先闭眼找感觉，我们都会有这样的经验，多眨眼可以让眼前稍微清晰一段时间，再一眨眼，这种清晰地感觉就没了，如果你能做到这一步，就说明你可以进入了第二阶段的练习，并尝试去延长这种感觉的时间吧。

第一步先微闭双眼，缓缓睁开。直视前方，然后双眼看着眼前的景象，快速将视线向中间靠近，就是将两眼所视焦点努力“聚焦”，（是看聚焦的物体，不是对眼，对眼“聚焦”太近了）然后你会感到额头及眼周的肌肉有很强的力量向里拉，这应该是眼部周在向里压眼球，之后会模糊一瞬间（非常短暂），接着绷住刚才有拉扯感觉的肌肉，眼前就清晰了一些（大概50度）。
