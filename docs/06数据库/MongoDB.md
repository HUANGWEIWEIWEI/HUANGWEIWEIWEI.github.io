## 使用
### 查看版本？
#### 1.用mongo命令启动数据库后输入
```
db.version()
```
#### 2.直接终端输入mongo 或者 mongod 命令
```
mongo --version
mongod --version
```
#### 3.还有一种就是使用命令登入系统时，返回的版本信息。
```
MongoDB shell version v3.6.8
connecting to: mongodb://.....
MongoDB server version: 3.6.8
Server has startup warnings:
```
